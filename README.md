# 3D_Rounded-OC-themes
Themes for use with OpenCore 0.7.0 and Higher and OpenCore Legacy Patcher (OCLP)


## First Use ➤  Change `config.plist` in `EFI/OC`

### Set PickerVariant (3D_Rounded_Hero theme for example)

- Misc —> Boot —> PickerVariant: Alphascorp\3D_Rounded_Hero

### Set PickerAtributes

- Misc —> Boot —> PickerAtributes: 145

Like this:

<img width="400" alt="1" src="https://github.com/alphascorp/3D_BSDrive-OC-Themes/blob/main/Screenshots/Screenshot_config.plist.jpg">
